API table:
Method Arguments Description
new playBook() name: string, field: PlayDisplayerObject, abletosave: bool, parentid: string Creates playBook object. Name needs to be unique. If creating playBook with no connection to field set field = null. parentid is deafult null and the object will be added to body if not specified
playBook().addpage() imagelink: string, playname: string, videolink: string, moveslist: list of strings Method to add page/play to playBook object. Moveslist is a list of valid moves for each player in the play. The order of the list is (LTE, LT LG, C, RB, RT, RTE, QB, LHB, FB, RHB). When set with a moveslist, the end user will be able to view the play animation if the book is connected to a playBook object. All arguments default to null.
playBook().allowUserCreatePlays() parentid: string Method that spawns a box which allows the end user to add plays to the playBook. If the playBook is connected to a PlayDisplayer, it will take the animations from there and store them in the book as well. parentid defaults null and if not specified object will be added to body
new PlayDisplayer() size: string, name: string, parentid: string Method that creates a PlayDisplayer (field) object which can be used to show animations. Must spercify unqique name and a size. The only sizes available are 'large' and 'xx-large'
PlayDisplayer().setLTEMove size: movename: string Method that sets the starting animation for the Left Tight End. Move must be a string from the valid move list (see below)
PlayDisplayer().setLTMove size: movename: string Method that sets the starting animation for the Left Tackle. Move must be a string from the valid move list (see below)
PlayDisplayer().setLGMove size: movename: string Method that sets the starting animation for the Left Guard. Move must be a string from the valid move list (see below)
PlayDisplayer().setCMove size: movename: string Method that sets the starting animation for the Center. Move must be a string from the valid move list (see below)
PlayDisplayer().setRGMove size: movename: string Method that sets the starting animation for the Right Guard. Move must be a string from the valid move list (see below)
PlayDisplayer().setRTMove size: movename: string Method that sets the starting animation for the Right Tackle. Move must be a string from the valid move list (see below)
PlayDisplayer().setRTEMove size: movename: string Method that sets the starting animation for the Right Tight End. Move must be a string from the valid move list (see below)
PlayDisplayer().setQBMove size: movename: string Method that sets the starting animation for the Quarterback. Move must be a string from the valid move list (see below)
PlayDisplayer().setLHBMove size: movename: string Method that sets the starting animation for the Left Halfback. Move must be a string from the valid move list (see below)
PlayDisplayer().setLTEMove size: movename: string Method that sets the starting animation for the Fullback. Move must be a string from the valid move list (see below)
PlayDisplayer().setLTEMove size: movename: string Method that sets the starting animation for the Right Halfback. Move must be a string from the valid move list (see below)
PlayDisplayer().spawnSandbox() save: bool, parentid: string Method that spawns a dropdown menu to allow the end user to view their own custom animations. When connected to a playBook object, set save to true when you want the end user to be able to add their plays they create to the playBook. Otherwise, set it to false or null. parentid defaults to null and the object will be added to the body if not specified.
